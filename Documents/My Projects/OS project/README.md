# Mini Shell OS

A simple Unix-like shell implemented for educational purposes. This project demonstrates basic shell functionalities such as command parsing, execution, and process management.

## Features
- Command execution (built-in and external)
- Input/output redirection
- Pipelining
- Basic job control
- Customizable prompt

## Getting Started

### Prerequisites
- GCC or compatible C compiler
- Unix-like environment (Linux, macOS, WSL)

### Build
```
gcc -o minishell minishell.c
```

### Usage
```
./minishell
```
